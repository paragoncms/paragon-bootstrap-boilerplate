Paragon CMS Sample Site

Bootstrap v3 Boilerplate

[[+Row]]
[[+Markdown]] Welcome to the Paragon CMS Bootstrap 3 Boilerplate project. This project represents a very simple example of a static site which can be used to build new sites! It includes a favicon and Bootstrap 3 and that's it. [[-Markdown]]

The source for this project is publicly available at Github. The project can be easily forked to create a new web site project of your own on Paragon CMS.

Bootstrap is a great way to make mobile responsive sites.

Check out their great documentation.
